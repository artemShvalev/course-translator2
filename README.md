
# 🎬 Video Subtitle Translator (en → ru)

Простая и быстрая программа для офлайн-перевода английских видео/аудио в русские субтитры (формат SRT).

✅ Работает полностью без интернета  
✅ Поддерживает пакетную обработку нескольких файлов  
✅ Отмена операций и прогресс-бар  
✅ Сохраняет настройки модели  
✅ Логирование ошибок  

---

## 🚀 Возможности

- Распознавание английской речи (Whisper)
- Перевод на русский язык (Argos Translate)
- Создание файла субтитров `_ru.srt` рядом с исходным
- Выбор модели Whisper (tiny, base, small, medium, large-v3, large-v3-turbo)
- Пакетная обработка нескольких видео/аудиофайлов
- Отмена долгой операции в любой момент
- Красивый тёмный интерфейс (sv-ttk)

> **Внимание:** в текущей версии нет синтеза речи и склейки с видео – только субтитры. Это сделало программу лёгкой и быстрой.

---

## 📦 Установка и запуск

### Для пользователей (exe)

1. Скачайте последнюю версию из [Releases](https://github.com/ВАШ_АККАУНТ/ВАШ_РЕПОЗИТОРИЙ/releases)
2. Распакуйте архив в любую папку
3. Запустите `VideoTranslator.exe`

> При первом запуске программа скачает модели Whisper (первые секунды) – это нормально.

### Для разработчиков (из исходников)

```bash
git clone https://github.com/ВАШ_АККАУНТ/ВАШ_РЕПОЗИТОРИЙ.git
cd ВАШ_РЕПОЗИТОРИЙ
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python gui_translator.py

Автономное приложение для перевода лекций, курсов и любых видео с английского на русский язык.  
Работает полностью без интернета, использует Whisper для распознавания, Argos Translate для перевода и Silero TTS для синтеза речи.

## ✨ Возможности
- Перевод видео/аудио в SRT субтитры
- Синтез русской озвучки (реалистичный голос, регулировка скорости)
- Склейка новой аудиодорожки с оригинальным видео
- Пакетная обработка нескольких файлов
- Отмена операций, прогресс-бар
- Сохранение настроек (модель, устройство, голос)
- Поддержка GPU (CUDA) для ускорения

## 📦 Установка и запуск

### Для пользователей (exe-файл)
1. Скачайте последнюю версию из [Releases](https://github.com/ВАШ_АККАУНТ/course-translator/releases).
2. Поместите `CourseTranslator.exe` в любую папку.
3. (Обязательно) Скачайте `ffmpeg.exe` с [официального сайта](https://ffmpeg.org/download.html) и положите в ту же папку.
4. Запустите `.exe`. При первом синтезе скачается модель Silero (около 200 МБ).

### Для разработчиков (из исходников)
```bash
